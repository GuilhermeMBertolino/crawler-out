Bridge Mode Help

Bridge mode: Set this router in bridge mode and use its 802.11ac WIFI connection to your existing router or gateway. This mode allows you to take advantage of gigabit WIFI speeds on current devices and avoid the need for separate WIFI adapters for each device. Some router functions are not available in bridge mode.

Router Functions

The following router functions are not available if you choose access point or bridge mode:

  • Guest network
  • Parental control
  • QoS
  • Access control
  • Traffic meter
  • Block sites
  • Block sites
  • Block services
  • Static route

Get IP Address Dynamically

Clear the Get IP Address Dynamically check box if you want to manually enter the IP address, subnet mask, and gateway IP addresses.

Get DNS Server Address Dynamically

Clear the Get DNS Server Address Dynamically check box if you want to manually enter the DNS server IP address.
