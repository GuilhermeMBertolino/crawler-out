Client Settings

This page allows the user to configure an dynamic dns account registered with the sever.

Configure dynamic dns account(s) with the details supplied.

By default, the page shows existing account informations. Click button to add a new account or user can modify or delete an existing account.

Dynamic dns account Account information consists of the dynamic dns server name , interface and host name associated with username and password.

More: Displays all the hostname associated with the user account.

Supported Servers

Shows the list of supported dynamic dns servers.By default supported servers are dhs, dyndns and dyns

By default, the page shows all the supported servers. Click button to add a new server or user can modify or delete an existing supported servers.

New servers can only be added if the servers are amongst the list of supported servers.
