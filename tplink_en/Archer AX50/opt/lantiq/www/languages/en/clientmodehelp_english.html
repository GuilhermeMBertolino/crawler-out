WiFi Repeater settings

This page allows you to view and configure wireless repeater.

Endpoint Enable Enable or Disable selected Radio tag Backhaul channel
WDS Enable or Disable backhaul connectivity using WDS method
Enable Repeater Mode Enable or Disable repeater mode
Wireless Network Names Pressing scan button will begin scanning for all available SSIDs on the selected Radio. User can also insert SSID value not found in the scan
Autentication Method Choosing the backhaul network Authentication method from drop down menu
SVAP Status Indication regarding SVAP connectivity status (Disconnected / Connected)
Repeater Management IP Allow user to choose what will be the management IP of the repeater after switching from GW to repeater is completed. User can choose to work with DHCP client or with a manually selected IP
Access Point Configure the repeater VAPs for each supported radio - User can choose to enable / disable each radio and what will be the published SSID
