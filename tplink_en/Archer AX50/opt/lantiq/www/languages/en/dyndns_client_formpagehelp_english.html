Client Settings

This page allows the user to add or edit clients of dynamic dns server.Username Interface and Server fields are mandatory field to add a client

Enable Enables or disables this Dynamic DNS Client.
Server The DynamicDNS server associated with this Dynamic DNS Client.
Interface The IP interface over which update queries to the server are sent.
Username Username used by this Dynamic DNS Client to authenticate with the Server.
Password Password used by this Dynamic DNS Client to authenticate with the Server.
Hostname Fully qualified domain name that has to be updated with the IP address.Multiple Hostname can be configured.
