ACL Filter

By enabling ACL feature, access control list can be configured based on address, port, protocols along with time. By default ACL feature is disabled. To apply the rules, the feature must be enabled.

Please note the following -
`The Subnet Mask field represents the length of subnet in case of IPv4 address or the length of prefix in case of IPv6 address.
`The time fields should be in the hh:mm:ss format.
`Appropriate type of address is a mandatory field in the configuration when the IP address is specified.
`Whenever a port is specified, the protocol is expected to be either UDP or TCP.
