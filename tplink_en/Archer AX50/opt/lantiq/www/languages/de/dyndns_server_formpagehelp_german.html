Server-Einstellungen

Auf dieser Seite kann der Benutzer dynamische DNS-Server hinzufügen oder bearbeiten. Servername Server-Adresse und Dienstname-Felder sind Pflichtfelder, um einen Server hinzuzufügen

Aktivieren Aktiviert oder deaktiviert Dynamic DNS Server.
Name Der Textname des dynamischen DNS-Servers
Dienstname Dynamischer DNS-Dienst, der verwendet wird, um diesen Server zu kontaktieren
Serveradresse Hostname oder IP-Adresse des Dynamic DNS-Servers.
