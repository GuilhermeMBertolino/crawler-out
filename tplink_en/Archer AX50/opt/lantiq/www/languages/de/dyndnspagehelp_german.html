Client-Einstellungen

Auf dieser Seite können Sie ein dynamisches dns-Konto konfigurieren, das mit dem Server registriert ist.

Konfigurieren Sie dynamische dns-Konten mit den angegebenen Details.

Standardmäßig werden auf der Seite vorhandene Kontoinformationen angezeigt. Klicken Um ein neues Konto hinzuzufügen oder ein bestehendes Konto zu ändern oder zu löschen.

Dynamisches dns-Konto Die Kontoinformationen bestehen aus dem dynamischen dns-Servernamen, der Schnittstelle und dem Hostnamen, die mit Benutzernamen und Kennwort verknüpft sind.

More: Zeigt den gesamten Hostnamen an, der dem Benutzerkonto zugeordnet ist.

Unterstützte Server

Zeigt die Liste der unterstützten dynamischen dns-Server an. Als standardmäßig unterstützte Server sind dhs, dyndns und dyns

Standardmäßig werden auf der Seite alle unterstützten Server angezeigt. Klicken Um einen neuen Server hinzuzufügen, oder Sie können bestehende unterstützte Server ändern oder löschen.

Neue Server können nur hinzugefügt werden, wenn die Server zu der Liste der unterstützten Server gehören.
