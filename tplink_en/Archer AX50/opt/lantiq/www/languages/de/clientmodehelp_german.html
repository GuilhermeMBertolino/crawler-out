WiFi Repeater Einstellungen

Auf dieser Seite können Sie den drahtlosen Repeater anzeigen und konfigurieren.

Endpunkt aktivieren Aktivieren oder deaktivieren Sie das ausgewählte Radio-Tag-Backhaul-Kanal
WDS Aktivieren oder Deaktivieren der Backhaul-Konnektivität mit der WDS-Methode
Wiederholungsmodus aktivieren Aktivieren oder Deaktivieren des Repeater-Modus
Drahtlosnetzwerknamen Durch Drücken der Scan-Taste beginnt das Scannen für alle verfügbaren SSIDs auf dem ausgewählten Radio. Benutzer kann auch SSID-Wert einfügen, der nicht im Scan gefunden wird
Authentifizierungsmethode Auswahl des Backhaul-Netzwerks Authentifizierungsmethode aus dem Dropdown-Menü
SVAP Status Hinweis zum SVAP-Konnektivitätsstatus (Disconnected / Connected)
Repeater Management IP Erlauben Sie dem Benutzer, zu wählen, was die Management-IP des Repeaters sein soll, nachdem das Umschalten von GW auf Repeater abgeschlossen ist. Benutzer kann wählen, mit DHCP-Client oder mit einer manuell ausgewählten IP zu arbeiten
Access Point Konfigurieren Sie die Repeater-VAPs für jedes unterstützte Radio - Der Benutzer kann das Radio aktivieren und deaktivieren und was ist die veröffentlichte SSID
