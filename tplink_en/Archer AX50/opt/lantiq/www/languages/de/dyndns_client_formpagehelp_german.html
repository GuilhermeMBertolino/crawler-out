Client-Einstellungen

Auf dieser Seite können Benutzer Clients von dynamischen DNS-Servern hinzufügen oder bearbeiten. Benutzernamen und Serverfelder sind Pflichtfelder, um einen Client hinzuzufügen

Aktivieren Aktiviert oder deaktiviert diesen dynamischen DNS-Client.
Server Der DynamicDNS-Server, der diesem dynamischen DNS-Client zugeordnet ist.
Schnittstelle Die IP-Schnittstelle, über die Aktualisierungsabfragen an den Server gesendet werden.
Benutzername Benutzername, der von diesem dynamischen DNS-Client für die Authentifizierung mit dem Server verwendet wird.
Passwort Kennwort, das von diesem dynamischen DNS-Client für die Authentifizierung mit dem Server verwendet wird.
Hostname Vollständig qualifizierter Domänenname, der mit der IP-Adresse aktualisiert werden muss. Mehrerer Hostname kann konfiguriert werden.
